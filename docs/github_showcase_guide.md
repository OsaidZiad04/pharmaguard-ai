# GitHub Showcase Guide

## Repository Description

Pharmacist-centered AI copilot prototype with privacy-safe OCR intake, local RAG, KB governance, retrieval diagnostics, safety-rule prompts, traceability, and synthetic evaluation.

## Suggested Pinned Project Text

PharmaGuard AI demonstrates a safety-first healthcare AI workflow: OCR output stays unverified, pharmacist correction is required, medication information is retrieved from local governed sources, and all counseling remains draft-only. The project emphasizes evaluation, governance, and pharmacist-in-the-loop design.

## Suggested README Badges

Use static badges only if desired:

- Prototype
- Pharmacist-in-the-loop
- Synthetic data only
- Local RAG
- No clinical validation

Avoid badges that imply production deployment, regulatory approval, or clinical validation.

## Brand Assets

Use assets from `frontend/public/brand/` in screenshots and demo material. In Next.js, reference them from public paths such as `/brand/pharmaguard-logo-main.png` and `/brand/pharmaguard-hero-visual.png`.
